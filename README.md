# Pig Game

A two-player dice game built with HTML, CSS and JavaScript.

## Live Demo

https://catalinalexandra.github.io/PigGame/

## Features

- Two-player gameplay
- Dice rolling
- Score tracking
- Hold score
- New game functionality

## Technologies

- HTML5
- CSS3
- JavaScript

AI tools (ChatGPT and Claude) were used as coding assistants to refine parts of the CSS styling and visual design. The project implementation and JavaScript logic were completed as part of my learning process.
